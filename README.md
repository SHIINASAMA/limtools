# limtools - 个人用工具包(Windows & Linux)

#### 包含功能

- IO相关类
  - 文件类 - **File class**
  - 控制台类 - **Console class**
  - Config配置文件类 **ConfigReader class & ConfigWriter class**
  - IO类（在Windows平台处理句柄）**IO class**
- 网络相关类
  - 广播类 - **Broadcast class**
  - UDP通信类 - **DatagrameSocket class**
  - UDP通信服务端类 - **DatagrameSocketServer class**
  - TCP通信类 - **Socket class**
  - TCP通信服务端类 - **SocketServer class**
- 线程相关类
  - 线程类 - **Thread class**
  - 安全线程类 - **SafeThread class**
  - 线程组类 - **ThreadGroup class**
  - 任务类 - **Task class**
  - 不可变线程池类 - **FixedThreadPool class**
